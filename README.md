### CPU RayTracer

Install
```
sudo apt install \
    libembree-dev \
    libglm-dev \
    libstb-dev \
    libassimp-dev \
    ffmpeg \
    ibboost-program-options-dev \
    libopenexr-dev \
    libimath-dev \
```

Run
```
./build-run.sh
```

or run demos in the `demos/` folder


##### Units

1 word unit = 1 meter
